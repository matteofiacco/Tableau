# Tableau

Il progetto viene illustrato con una presentazione PowerPoint in cui è presente il link della mia visualizzazione su Tableau Public.

L'analisi si concentra nello studio del palinsesto di Netflix, utilizzando un dataset pubblico di Kaggle in cui sono presenti i dettagli 
di ogni show: titolo, regista, cast, Paese di produzione, anno di uscita e rating.

L'obiettivo è stato quello di ottenere insight interessanti per proporre una strategia per uno show.
